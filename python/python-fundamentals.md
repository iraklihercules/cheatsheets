# Python Fundamentals

1. Data Types
```python
bool
int
float
str
list    # []
tuple   # () - Tuples are immutable.
set     # {"key"} - Set can't store duplicated values.
dict    # {"key": "value"}
```

Operators
```python
print(2 ** 2)  # 2 power of 2
print(2 ** 3)  # 8
print(2 ** 4)  # 16

print(2 / 4)   # (float) 0.5
print(2 // 4)  # (int) 0

print(5 % 4)   # (modulo) 1
print(7 % 4)   # (modulo) 3

# Operators precedence
()
**
*  /
+  -
```

Debugging
```python
type(4)  # <class 'int'>
```

Math Functions
```python
round(3.555)     # 3
round(3.555, 2)  # 3.56
abs(-20)         # 20
```
